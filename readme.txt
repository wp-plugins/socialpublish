SocialPublish is an easy to use service that automatically shares your blog post on Facebook and Twitter at the moment you publish it.

== Description ==

For detailed description of using and installing the SocialPublish WordPress plugin, 
please visit our [SocialPublish WordPress plugin documentation page](http://socialpublish.io/plugins/wordpress "SocialPublish WordPress plugin documentation page").


== Screenshots ==

1. When you write a blog post and have the SocialPublish Screen Option panel enabled, you're able to provide optional hashtags and a message text you want to provide within your published Twitter and/or Facebook post.
2. Within the SocialPublish Dashboard you're able to add, enable, disable and remove social media platforms.
